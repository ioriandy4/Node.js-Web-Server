在html-test目录安装http-server，  npm install
然后执行http-server -p 8001，就会运行一个以当前目录为根目录的服务器

接口是8000
静态文件是8001
nginx是8080
nginx配置反向代理
location / {
        proxy_pass http://localhost:8001;
        index  index.html index.htm;
}

location /api/ {
        proxy_pass  http://localhost:8000;
        proxy_set_header Host $host;
}
